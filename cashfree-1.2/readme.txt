Cashfree Payment plugin for WooCommerce!

== Description ==

Cashfree is a fast-growing payment gateway, we serve businesses in India and help them accept online payments over various payment channels. We have direct integrations with multiple banks which helps us in offering a fast and reliable payment gateway. You can use this plugin to activate Cashfree Payment gateway on your WooCommerce site. For more details visit our [website](https://cashfree.com).
 - Accept payments in more than 100 modes - Credit Cards, Debit Cards, Net Banking, UPI, Wallets (PayTM, Mobikwik, etc.), EMI & pay later options.
 - Accept International Cards
 - Integration with Paypal
 - Settlement in under 48 hours
 - Opt-in for same day refund processing and keep your customers happy
 - and many other features. See more at [Cashfree](https://cashfree.com)
 - You would need a Cashfree account to use this Plugin. Sign-Up [here](https://merchant.cashfree.com/merchant/login#signup)

== Installation ==

You can install the plugin through the Wordpress plugins screen directly. Alternatively you can also-

1. Upload the plugin files to the `/wp-content/plugins/cashfree` directory
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Cashfree screen to configure the plugin
